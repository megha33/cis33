Node.JS!

Introduction

Node.JS is an open source source environment that uses Javascript on the server. It uses Chrome's V8 engine and is event-driven. It's commonly used create dynamic page content, collect data, and manipulate files & items in the database. Before you use Node, it MUST be initiated on the server otherwise it won't have any effect.

Applications that use Node.JS

  • Ebay
  • Quizlet
  • Netflix
  • Paypal
  • LinkedIn
